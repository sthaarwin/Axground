# Axground Next.js App

This is the automated implementation of the **Axground / NEPSE Market Dashboard** Stitch project.

## Features
- **Market Dashboard**: Real-time NEPSE index and sentiment tracking.
- **RL Training Lab**: Live monitoring of reinforcement learning agents (PPO/DQN).
- **Strategy Comparison**: Side-by-side performance analysis of multiple models.
- **Backtesting Suite**: Historical simulation on NEPSE data.

## Built With
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **Components**: Shadcn/ui (Radix UI)
- **Icons**: Lucide React
- **Theme**: Premium Dark Mode (#136dec primary)

## Getting Started

1. **Navigate to the directory**:
   ```bash
   cd axground
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Access the platform**:
   Open [http://localhost:3000](http://localhost:3000) in your browser.
